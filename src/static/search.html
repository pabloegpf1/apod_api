APOD API - search endpoint example

What's this endpoint for?

This endpoint is for fetching APODs that match requested query string.

How to use it?

The API's endpoint is: .

Available query parameters:

  • search_query: (required!) a string, the API will search for APODs with this string.
  • number: optional, but recommended integer, the API will return only n number of APODs that match the search_query.
  • page: an integer, the API will return nth page of APODs if number is specified.
  • image_thumbnail_size: (optional) an integer, when larger than 0, image_thumbnail will be returned in the JSON file (see below).
  • absolute_thumbnail_url: (optional) a boolean, when set to true, image_thumbnails will have absolute, not relative URLs.
  • thumbs: (optional) a boolean, when set to true, a video thumbnail will be returned if the APOD is a video. Can be used in list requests.
  • html_tags: (optional) a boolean, when set to true, the description will be in the original HTML format. Can be used in list requests.
